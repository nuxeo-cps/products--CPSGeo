__bobo_traverse__(self, REQUEST, name)

Hook for Zope 2 traversal

This method is called by Zope 2's ZPublisher upon traversal. It allows us to trick it into faking the Zope 3 traversal system by using an ITraverser adapter.

__fallback_traverse__(self, REQUEST, name)

Method hook for fallback traversal

This method is called by __bobo_traverse___ when Zope3-style ITraverser traversal fails.

Just raise a AttributeError to indicate traversal has failed and let Zope do it's job.
