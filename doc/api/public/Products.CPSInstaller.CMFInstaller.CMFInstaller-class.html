Method Details

__init__(self, context, product_name=None)
(Constructor)

CMFInstaller initialization

context is used to pass the portal root.

product_name should be set as a class attribute when subclassing, but must be passed if you are not subclassing the installer.

_v_main_installer should be se to 0 if this installer is called from another installer to prevent multiple reindexing of catalogs and similar actions that only needs to be done once.
