Method Details

afterClear(self)

Called after the fixture has been cleared. Note that this may occur during setUp() *and* tearDown().

afterSetUp(self)

Called after setUp() has completed. This is far and away the most useful hook.

beforeClose(self)

Called before the ZODB connection is closed, at the end of tearDown(). By default aborts the transaction.

beforeSetUp(self)

Called before the ZODB connection is opened, at the start of setUp(). By default begins a new transaction.

beforeTearDown(self)

Called before tearDown() is executed. Note that tearDown() is not called if setUp() fails.

logout(self)

Logs out.

setUp(self)

Sets up the fixture. Do not override, use the hooks instead.
