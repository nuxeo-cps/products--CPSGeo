Constructor Detail

ButtonBase

ButtonBase(button, widgetNode, model)
    Abstract base Button object that all Buttons extend. A Button is a widget which renders an image and an optional second image for the enabled state.
    Parameters:
      button - Pointer to the button instance being created.
      widgetNode - The tool node from the Config XML file.
      model - The parent model object (optional).
    Author: Mike Adair mike.adairATccrs.nrcan.gc.ca
Method Detail

buttonInit

void buttonInit(objRef)
    Initialise image for the button and select it if required.
    Parameters:
      objRef - Reference to this object.

doAction

void doAction()
    Override this in buttons which inherit from this object to carry out the action. This is the function that will be called either when the button is selected via the select() method or on a mouseup event if there is an associated mouseHandler property in config.

doSelect

void doSelect(selected, objRef)
    Override this function in Buttons to process select/deselect calls.
    Parameters:
      selected - True when selected, false when deselected.
      objRef - Reference to this object.

initMouseHandler

void initMouseHandler(objRef)
    A listener method to initialize the mouse handler, if configured. This is called as an init event so that the object pointed to is guaranteed to be instantiated.
    Parameters:
      objRef - Reference to this object.

prePaint

void prePaint(objRef)
    Override of widgetBase prePaint to set the doc to be styled as the widget node in config for this button.
    Parameters:
      objRef - Pointer to this object.

select

void select()
    Called when a user clicks on a button. Switches the image to the enabled button source, enables and disables associated tools, then calls the doSelect method defined in derived classes.
