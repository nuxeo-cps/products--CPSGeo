Context

Context(modelNode, parent)
    Stores a Web Map Context (WMC) document as defined by the Open GIS Consortium http://opengis.org and extensions the the WMC. Listeners supported by this model: "refresh" called when window parameters (width/height, bbox) are changed "hidden" called when visibilty of a layer is changed
    Parameters:
      modelNode - Pointer to the xml node for this model from the config file.
      parent - The parent model for the object.
    Author: Cameron Shorter
