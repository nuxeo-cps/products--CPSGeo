mouseOverHandler

void mouseOverHandler(objRef, targetNode)
    Start cursor tracking when the mouse is over the MapContainer. set an interval to output the coords so that it doesn't execute on every move event. The setInterval method in IE doesn't allow passing of an argument to the function called so set a global reference to MapContainer here; mouse can only be over one MapContainer at time so this should be safe.
    Parameters:
      objRef - Pointer to this CurorTrack object.
      targetNode - The node for the enclosing HTML tag for this widget.
