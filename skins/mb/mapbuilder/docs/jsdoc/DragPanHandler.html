DragPanHandler

DragPanHandler(toolNode, model)
    Tool to click and drag a map pane to achieve a recentering of the map. This tool processes screen coordinates and stores AOI in the current map projection coordinates.
    Parameters:
      toolNode - The tool node from the Config XML file.
      model - The model object that contains this tool.
