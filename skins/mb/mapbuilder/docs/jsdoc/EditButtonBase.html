doSelect

void doSelect(selected,objRef)
    If tool is selected and the Edit Tool has changed (eg, changed from LineEdit to PointEdit) then load new default feature. This function is called when a tool is selected or deselected. The following parameters are copied from this button's config node into the target model:
    "transactionResponseModel", "webServiceUrl", "featureXpath", "defaultModelUrl", "targetContext". These values will be used by WebServiceAction when processing a transaction.
    Parameters:
      selected - True when selected.
      objRef - Pointer to this object.
