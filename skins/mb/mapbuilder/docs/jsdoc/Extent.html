Constructor Detail

Extent

Extent( model, initialRes )
    A tool designed to handle geography calculations for widgets which render the model in 2D. Use of this tool requires that it's model implements get/setWindowHeight/Width methods. Encapsulates all geography and image size aspects of a geographic object displayed in a rectangular area on the screen. All coordinates are handled as points which is a 2 element array, where x is the first element and y is the second. Coordinates are either pixel and lixel (pl) relative to the top left of the extent or projection XY values (xy).
    Parameters:
      model - the model document that this extent represents
      initialRes - (optional) if supplied the extent resolution will be set to this value

Object centerAt(center, newres, limitExtent)
    Adjust the extent so that it is centered at given XY coordinate with given resolution. Extent width and height remain fixed. Optionally check to ensure that it doesn't go beyond available extent.

      newres - resolution to display at
      limitExtent - ensure that the extent doesn't go beyond available bbox (TBD: not complete/tested)
