OwsContext

OwsContext(modelNode, parent)
    Stores an OWS Context document as defined by the OGC interoperability experiment. This model should be eventually merged with the standard OGC context doc. Listeners supported by this model: "refresh" called when window parameters (width/height, bbox) are changed "hidden" called when visibilty of a layer is changed "wfs_getFeature" called when feature resources are loaded
    Author: Mike Adair
    Requires:
      - Sarissa
