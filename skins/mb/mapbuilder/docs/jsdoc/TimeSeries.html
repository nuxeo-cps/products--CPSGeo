Constructor Detail

TimeSeries

TimeSeries(widgetNode, model)
    Widget to render a time series of maps from an OGC context document. Each map layer is rendered one on top of the other and only the first one is visible. the MovieLoop tool is used to cycle the visibility of the layers. The timstamping is handled as an array, ie the array index is used to select layers.
    Parameters:
      widgetNode - The widget's XML object node from the configuration document.
      model - The model object that this widget belongs to.
