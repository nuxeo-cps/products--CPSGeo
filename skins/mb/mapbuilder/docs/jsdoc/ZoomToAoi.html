ZoomToAoi

ZoomToAoi(toolNode, model)
    Controller for the locator map widget. Specify the context that this widget follows by setting the targetModel property in config. This will display the AOI of the target model using the AoiBox tool. This will also process mouse events (click and dragbox) to recenter the target model and includes coordinate projection transformations if required. Checking for extent limits is not yet implemented.
    Parameters:
      toolNode - The tool node from the config document for this tool
      model - Reference to the widget object that creates this tool
    Author: Adair
